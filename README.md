# p-modal

Using imperative management of modals in Vue

## Installation

```bash
# with npm
npm install p-modal

# with yarn
yarn add p-modal

# with pnpm
pnpm add p-modal
```

## Usage

```ts
import { showModal } from 'p-modal'
import MyModal from './MyModal.vue'

showModal(TestModal, { count: 1 })
```
